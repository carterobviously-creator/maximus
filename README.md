# Maximus - Angry Birds Clone 🎮

An epic Angry Birds clone built with HTML5, JavaScript, TypeScript, and Matter.js physics engine!

## Features

- 🎯 **Physics-Based Gameplay** - Realistic physics using Matter.js
- 🎵 **Background Music** - YouTube video playing in the background
- 🐦 **Slingshot Mechanics** - Drag and release to launch birds
- 🐷 **Destroy the Pigs** - Knock down structures and defeat all pigs
- 💥 **Particle Effects** - Explosion effects when hitting targets
- 🏆 **Scoring System** - Earn points for destroying pigs
- 🌈 **Random Thing™** - Secret retro mode with glitch effects (Press SPACE or click the rainbow button!)
- 📊 **HUD Display** - Score and birds remaining counter

## Game Elements

### Materials
- **Wood Blocks** (Brown) - Light and easy to destroy
- **Stone Blocks** (Gray) - Heavier and more durable
- **Glass Blocks** (Light Blue) - Fragile but transparent

### Controls
- **Click and Drag** the bird on the slingshot
- **Release** to launch
- **R Key** - Restart the game
- **SPACE Key** - Toggle retro mode (the "random thing")

## How to Play

1. Open `game.html` in your web browser
2. Click and drag the red bird backward on the slingshot
3. Release to launch the bird at the structure
4. Destroy all green pigs to win!
5. You have 3 birds to complete the level

## Technology Stack

- **HTML5 Canvas** - Game rendering
- **Matter.js** - 2D physics engine
- **JavaScript/TypeScript** - Game logic
- **CSS3** - Styling and animations
- **YouTube API** - Background music integration

## Files

- `game.html` - Main game page with UI and layout
- `game.js` - Compiled JavaScript game engine
- `game.ts` - TypeScript source code for the game engine
- `index.html` - Original MusicGen AI music generator app

## The "Random Thing" 🌈

Press SPACE or click the rainbow button at the bottom left to activate **Retro Mode**:
- Psychedelic visual effects
- Glitch animation
- Enhanced contrast and color shifts
- Pure nostalgia vibes!
